Statistical Modeling

Statistical Modeling can be used to build, design, and run a wide variety of statistical and machine learning models for both raster and vector type data. Examples of analyses that can be performed include: variance covariance, T-Tests, PCA, K-means clustering, linear, multivariate, logistic, polytomous logistic, accuracy assessment, neural networks, and random forest. These tools integrate Accord, Aforge, and ALGLIB .NET libraries directly into ArcMap. Many of the statistical tools need to be manually added to ArcMap via the customization tool.
