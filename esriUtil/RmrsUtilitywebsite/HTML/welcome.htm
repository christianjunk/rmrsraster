RMRS Raster Utility

An object oriented library packaged as an ESRI add-in that simplifies data acquisition, raster sampling, and statistical and spatial modeling while reducing the processing time and storage space associated with raster analysis. To find out more about the project, download the application, source code, tutorials, and demonstrations, or contact us please use the links in the navigation panel to the left.
