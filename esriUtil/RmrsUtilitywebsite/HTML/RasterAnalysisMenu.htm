Raster Processing

Most raster spatial analysis procedures can be accessed through the Raster Analysis Menu. Functionality includes:

  • Arithmetic Analysis
  • Surface Analysis (e.g., slope and aspect)
  • Math Transformation
  • logical Analysis
  • Conditional Analysis
  • Remap Raster
  • Focal Analysis
  • Convolution Analysis
  • Local Stats Analysis
  • Perform a linear Transform
  • Clip Analysis
  • Convert Pixel Type
  • Rescale Analysis
  • Create Composite
  • Extract Bands
  • Merge Rasters
  • Create GLCM
  • Landscape Metrics
  • Function Modeling
  • Zonal Statistics
Tools for each analysis work with multiband raster datasets and outputs are stored as Function Datasets with a pixel depth of 32 bit floating precision (exception being the Convert Pixel Type tool). Additional spatial analysis tools can be loaded into the toolbar using ArcMap's customization functionality.
