Help Menu

The Help menu contains 3 buttons that can be used to view documentation, the version of the RMRS Raster Utility toolbar, and the RMRS Raster Utility website.

Documentation opens the chm file associated with the RMRS Raster Utility. Upon opening, the toolbar check to see if the local help version matches the online help version. If the two do not match the user will be prompted if they want to update the help. If it is covenant and you have internet connection select yes.
Version opens a window identifying the version of the toolbar.
Opens the RMRS Raster Utility website.
