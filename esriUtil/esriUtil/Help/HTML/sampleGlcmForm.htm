Sample GLCM

This form is used to sample GLCM values given a series of sample locations. Currently some of the GLCM metrics create conventional model outputs as opposed to function datasets, which can substantial increase processing time. This tool allows the user to sample some of those metrics without creating the conventional data source, providing a way to test if those values warrant creating the full dataset. Future release of RMRS Raster Utility will deprecate this form and convert those metrics that produce conventional raster into function datasets. At that time users can use the Create GLCM Raster analysis tool to create function model and then sample that newly created function model using the Sample Raster Value form.

Sample Feature Class dropdown and open button: Allows users to select the point feature class that represents the sample locations.
Raster dropdown and open button: Allows the user to select which raster will be used to calculate GLCM surfaces.
Window Type dropdown: The type of analysis window used to look at neighboring raster values (Rectangle or Circle).
Columns, Rows, and Radius numeric dropdown: Allows users to specify the size of the analysis window. If window type is Circle the Columns and Rows numeric dropdown disapears and the Radius numeric dropdown appears. Values are number of cells within the window.
Direction dropdown: Allows users to select the direction of comparison for the GLCM (horizontal or vertical).
Types dropdown: Allows users to select the types of GLCM metrics to calculate. Once selected in the dropdown the users must click on the + button to add the metric to list of GLCM metrics to sample.
+ button: Adds metric types to the list of GLCM values to sample.
- button: Subtracts the currently selected metric type from the list of GLCM values to sample.
% button: Adds all metric types to the list of GLCM values to sample.
! button: Subtracts all metrics types from the list of GLCM values to sample.
Sample button: Samples the specified GLCM metric values for the specified raster surface.

Output

A .csv file located at the specified location.
